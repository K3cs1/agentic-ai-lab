# Behavior shaping with system messages

Using a system message to influence assistant behavior. This version includes an example of how to 
use a system message to set the behavior of the assistant, allowing for more control over the responses
generated by the chatbot.

By running the same test several times is also easy to notice that for the same prompt the responses vary.

# Prerequisites

- Ollama installed and running
- qwen2.5 model pulled locally
